我所使用的版本是python3.6.

数据处理分类

数据的处理大致分为四种情况【1】：

  1. 完整性：单挑数据-完整？空值？
  2. 全面性：每一列数据-定义？单位？本身？
  3. 合法性：类型？内容？大小？的合法性
  4. 唯一性：数据是否存在重复

此处需要给数据类型一个简单的介绍：

series:

一组数组（列表或元组），series除了一组数据外还包括一组索引（即只有行索引），索引可自行定义也可利用Series(),自动生成索引；

dataframe:

是表格型数据，既有行索引又有列索引，每列数据可以为不同类型数据（数值、字符串、布尔型值），可利用DataFrame（其他数据，dataframe属性)指定dataframe的属性创建dataframe。
