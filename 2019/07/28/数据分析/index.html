使用python采集数据

  1. 找到想要获取数据的网站
  2. 审查元素 -> network -> XHR -> 刷新 -> 查看请求得到对应路径
  3. 打开路径 -> 如果是JSON格式对象可以直接使用，python里面可以直接把JSON格式的对象转换成 python对象
  4. 但是有的时候，网页会用 JS 请求数据，那么只有都加载完之后，我们才能获取完整的 HTML 文件 -> XPath 可以不受加载的限制，帮助我们定位想要的元素。

JSON： 仅仅是一种数据格式，可以跨平台数据传输，速度快。

JS：表示类的实例，不能传输。

数据清洗

清洗规则：

  1. 完整性：单挑数据-完整？空值？
  2. 全面性：每一列数据-定义？单位？本身？
  3. 合法性：类型？内容？大小？的合法性
  4. 唯一性：数据是否存在重复

清洗工具：Pandas

具体的操作见另一篇博文

数据集成

eg: 随机挑选几位明星，粉丝总和达到20亿。这显然是不可能的，里面肯定存在着重复的数据 - 此时需要用到数据集成来解决问题。

两种架构

Extract, Transform, Load

  • ELT: 数据源抽取后先进行转换，然后将转换的结果写入目的地

  • ETL: 抽取后先将结果写入目的地，然后利用数据库的聚合分析能力或者外部计算框架，来完成转换步骤。

Extract: 将数据从已有的数据源中提取出来

数据变换

-让渠道不同的数据到一个统一的数据库里

-简单的理解就是数据在同样的范围中，拥有同样的计数方式，那么两者的拿到一起来进行数据分析才会有意义。

过程

字段过滤 -> 数据探索 -> 相关性分析 -> 建模筛选 -> 数据变换
